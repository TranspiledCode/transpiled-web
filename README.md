## Development

To set up and run the project locally, follow these steps:

1. **Clone the repository:**

   ```sh
   git clone git@github.com:TranspiledCode/transpiled-web.git
   ```

2. **Navigate to the project directory:**

   ```sh
   cd transpiled-web
   ```

3. **Install dependencies:**

   ```sh
   yarn install
   ```

4. **Run the development server:**

   To start the local server, use:

   ```sh
   yarn dev
   ```

   This command will run the development server using Parcel, allowing you to access the project at `http://localhost:2024` or through your local IP address.

## Build

1. **Build the project for production:**

   To create a production build, use:

   ```sh
   yarn build
   ```

   This will generate the optimized files in the `dist` folder, ready for deployment.
